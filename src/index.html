Pokédex

A Pokédex (palavra comum no vocabulário dos amantes de Pokémon) é uma enciclopédia virtual que detém todas as espécies de pokémon. Nesse projeto pude voltar um pouco no tempo e sentir a nostalgia da infância mais uma vez. Uma Pokédex com JavaScript, HTML, CSS coletando dados da conhecida API entre os devs, PokeAPI.

RoboFriends

Robofriends foi um dos primeiros projetos que realizei colocando em prática alguns dos conhecimentos em React. A construção também incluiu HTML5 e CSS3 e a utilização deste último foi bastante reduzida pelo uso do Tachyons. A ideia é bem simples e consiste em criar um grid com cards de robôs com imagens geradas unicamente pela Robohash e com um array de dados coletados através do site JSONPlaceholder.
